# C语言编译成汇编分析
C语言代码main.c:

```
int g(int x)
{
        return x + 99;
 }
 
 int f(int x)
 {
        return g(x);
 }
 
 int main(void)
 {
        return f(3) + 2;
 }
```
通过gcc编译成汇编

```
gcc –S –o main.s main.c -m32
```
